# 🎁 Amigo Secreto

O **Amigo Secreto** é um projeto interativo desenvolvido com **HTML, CSS e JavaScript**, permitindo que usuários adicionem participantes e realizem o sorteio de forma justa, garantindo que ninguém tire a si mesmo.

---

## 📖 Sobre o Projeto
Este projeto foi criado para facilitar a realização de sorteios de **Amigo Secreto** de maneira rápida e intuitiva. Ele inclui validações para evitar nomes duplicados e garantir um sorteio justo.

---

## 🚀 Tecnologias Utilizadas
- **HTML5** → Estruturação da página  
- **CSS3** → Estilização do layout  
- **JavaScript (ES6)** → Lógica de adição de amigos e sorteio
